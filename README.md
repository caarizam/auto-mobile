# Automation example with Appium, Android and Cucumber

## Requirements
- Appium >= **v1.8.1**
- Android Studio **>= 3.3.1**
- JDK >= **1.8**
- Apache Maven >= **3.5.4**

## Execution
- open appium in a console
- open Android Studio and open a AVD
- execute ```mvn clean```
- execute ```mvn install```
